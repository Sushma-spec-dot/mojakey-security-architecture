
## Project Summary
Mojakey is a security-focused system design portfolio project for a multi-tenant family and institution platform. The project demonstrates how to design a secure check-in/check-out system with role-based access control, audit logging, QR security, privacy protections, and trusted attendance and billing event integrity.

## Why this project matters
This project shows practical understanding of:
- secure system design
- application security
- access control
- privacy and data protection
- auditability and traceability
- multi-tenant isolation
- secure event workflows

## Security topics covered
- Authentication
- Authorization and RBAC
- Tenant isolation
- Audit logging
- QR code security
- Privacy protection
- Event integrity
- Incident investigation readiness
